jQuery Bing Search API Plugin

By Chris Benard

Information

This is an example page for jQuery-bingsearch, a jQuery plugin which allows you to search the Windows Azure Marketplace Bing Search API.

The plugin was created by Chris Benard and is available under the MIT license on GitHub.
